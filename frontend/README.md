# WoodPlan Events - Frontend 🎸

Este é o frontend da WoodPlan Events, desenvolvido com **Angular 19** e **TailwindCSS 4**.

## 🚀 Tecnologias
- **Angular 19**: Utilizando a nova sintaxe de controlo de fluxo (@if, @for).
- **TailwindCSS 4**: Estilização moderna e eficiente.
- **Lucide Angular**: Ícones elegantes.
- **Supabase JS**: Integração com autenticação e storage.

## 🛠️ Desenvolvimento

### Instalação
```bash
npm install
```

### Servidor de Desenvolvimento
```bash
npm start
```
O comando `npm start` executa automaticamente o build do Tailwind CSS antes de iniciar o servidor Angular.
Aceda a `http://localhost:4200`.

### Backoffice
Aceda a `http://localhost:4200/admin` para a área de gestão.

## 📦 Build para Produção
```bash
npm run build
```
Os ficheiros compilados serão guardados na pasta `dist/frontend`.
